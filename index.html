Emotify

A JavaScript library that lets you add custom emotes into your website.

Getting started with Emotify is a piece of cake!
1. Include the Emotify script.
<script src="https://cdn.jsdelivr.net/gh/aryanbaburajan/emotify/src/emotify.js"></script>
2. Create custom emotes.
emAddEmote({ "grin": "./grin.svg" })
3. Create an element with the em class, and Emotify will automatically convert the text into emojies for you.
<p class="em">Hello, World! :grin:</p>
Hello, World! :grin:
