emAddEmote()

emAddEmote() adds to the list of emotes, an emote that will be considered while emotify-ing text.

Syntax

emAddEmote({"emoteName", "emoteURL"});

Parameters

emotes An object containing the emote names and URLs as key and value pairs.

Example

emAddEmote({
	"sigh": "./sigh.png",
	"sweat": "./sweat.png",
	"neutral": "./neutral.png",
	"love": "./love.png",
	"sleepy": "./sleepy.png",
});

emConvert()

emConvert() Reconverts all texts into emotes, useful in scenarios where the text changes after page load.

Syntax

emConvert();

Parameters

No parameters.

Example

emConvert();
